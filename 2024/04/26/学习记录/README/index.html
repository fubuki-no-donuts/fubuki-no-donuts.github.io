  • UE5：高级 UI 小工具

    • 第一部分
      • 005 运行时导入数据：C++ 创建 UBlueprintFunctionLibrary 类用于导入数据，这里会声明成 static 函数，需要用到Misc/FileHelper.h，IPlatformFile：文件的 I/O 接口，IPlatformFile& PlatFormFile = FPlatformFileManager::Get().GetPlatformFile();
      • 006 运行时解析数据：利用之前创建的函数库，在一个空 Actor 的事件里实现了对 csv 文件的处理（教程中为道具名称、价格等等），并把这个 actor 拖入了场景里。演示了一下在 csv 文件里修改数据后可以被动态读取，但演示中只是游戏开始运行时才读取
      • 007 AssetRegistry 资产注册表，可以利用这个注册表在运行时从任何地方获取资产，获取这个注册表的方式GetAssetRegistry，一个纯函数
      • 008 数据驱动的小部件系统：从外部 csv 文件读取格式化数据，其实用 datatable 就可以实现，这里应该是为了做教程才这么讲的
    • 第二部分
      • 01 Overriding Events And Functions：在 widget 里重载一些事件时注意返回的 handled 还是 unhandled，handled 会消耗当前的输入事件，例如鼠标的点击。后者则不会消耗，意味着如果两个控件重叠出现，返回 unhandled 会导致下层控件也会对鼠标点击事件产生响应
      • 02 DragAndDrop Widget：有一个继承自UObject的DragDropOperation，说明里表示这个是 UMG 的基础拖放操作。这里的东西有意思，之前没了解过
      • 03 预构造事件：这里提到了在一个控件容器里构造小控件之前，先把容器 clear 一下
      • 04 ControllerMapping：这里也看一下，控制器在控件上移动，核心是确认HasAnyUserFocusDescendants为真的控件设置为Hovered
